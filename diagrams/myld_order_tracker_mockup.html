New order entry
1. Receive PO from Sales rep via email or CRM.  2. Enter into ERP (Business Central) within 24hrs.  3. Confirm SKU, qty, and state compliance requirements.  4. Set status to Pending and notify distribution.
Order rejection / refusal
1. Log in Discrepancy log with issue type and description.  2. Update ERP status to Rejected.  3. Notify Sales rep and Accounting same day.  4. Issue credit memo if delivery was previously invoiced.
ERP / CRM mismatch resolution
1. Flag in Discrepancy log with "ERP/CRM mismatch" type.  2. Cross-reference with distribution's delivery confirmation.  3. Correct the lagging system and document resolution.  4. Notify Accounting if invoice status is affected.
